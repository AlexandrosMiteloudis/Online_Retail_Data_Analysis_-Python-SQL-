# Online Retail Data Analysis (Python + SQL)

## Project Overview
This project explores the Online Retail Dataset from Kaggle using Python for data cleaning and exploratory data analysis (EDA), followed by SQLite for relational querying and SQL practice.

## Objectives
- Clean and preprocess transactional retail data
- Handle null values, duplicates, cancellations, and incorrect entries
- Perform exploratory data analysis with Python
- Store cleaned data in SQLite
- Practice SQL querying for business insights

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQLite

## Dataset
Source: Kaggle Online Retail Dataset (https://www.kaggle.com/datasets/ulrikthygepedersen/online-retail-dataset)

## Workflow
1. Data Loading
2. Data Cleaning
